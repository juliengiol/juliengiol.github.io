Let \(\mathfrak{g}\) be a Lie algebra over a commutative ring \(R\text{.}\)

\begin{equation} \Lambda^3\mathfrak{g}\stackrel{\partial_3}{\longrightarrow}\Lambda^2\mathfrak{g}\stackrel{\partial_2}{\longrightarrow}\mathfrak{g}\tag{3.1.1} \end{equation}
\begin{equation} \partial_2(x\wedge y)=[x,y]\tag{3.1.2} \end{equation}
\begin{equation} \partial_3(x\wedge y\wedge z)=[x,y]\wedge z+[y,z]\wedge x+[z,x]\wedge y\tag{3.1.3} \end{equation}
Remark 3.1.1.

Given these definitions, the Jacobi identity boils down to

\begin{equation} \partial_2\circ\partial_3=0.\tag{3.1.4} \end{equation}

The \(R\)-module

\begin{equation} H_2(\mathfrak{g};R)=\mathrm{Ker}\,\partial_2/\mathrm{Im}\,\partial_3\tag{3.1.5} \end{equation}

Now assume that \(\mathfrak{g}\) has finite dimension over \(R\) and fix a basis \(\{e_1,\ldots,e_n\}\) so that

\begin{equation*} \mathfrak{g}=Re_1\oplus\cdots\oplus Re_n. \end{equation*}

Then all the information about \(\mathfrak{g}\) is contained in the \(n^3\) structure constants

\begin{equation*} \left\{c_{i,j}^k\,|\,1\le i,j,k\le n\right\} \end{equation*}

which are defined by the following \(n^2\) equations:

\begin{equation} [e_i,e_j]=\sum_{k=1}^nc_{i,j}^ke_k\qquad\text{for all}\,1\leq i,j\le n.\tag{3.1.6} \end{equation}

If we equip \(\Lambda^2\mathfrak{g}\) with the basis

\begin{equation*} \left\{e_i\wedge e_j\,|\,1\le i<j\le n\right\}, \end{equation*}

then the matrix of \(\partial_2\) is

\begin{equation*} D_2=\left[c_{i,j}^k\right] \end{equation*}

with rows indexed by \(k\) and columns indexed by \((i,j)\text{.}\)

Example 3.1.2.

Lemma 3.1.3.

Let us consider the set

\begin{equation*} S=\{(i,j)\,|\,1\leq i<j\leq n\}. \end{equation*}

Then

\begin{equation*} |S|=\binom{n}{2} \end{equation*}

and if we list the elements of \(S\) in lexicographical order, we have

\begin{equation*} S=\{\underbrace{(1,2)}_{s_1},\underbrace{(1,3)}_{s_2},\ldots,\underbrace{(1,n)}_{s_{n-1}},\underbrace{(2,3)}_{s_n},\ldots,\underbrace{(i,j)}_{s_u},\ldots,\underbrace{(n-1,n)}_{s_{|S|}}\} \end{equation*}

with

\begin{equation*} u=\binom{n}{2}-\binom{n-i+1}{2}+j-i. \end{equation*}
Proof.

First observe that if we denote

\begin{equation*} (r,s)<(i,j) \end{equation*}

the lexicographical order, then

\begin{equation*} u=\left|\{(r,s)\in S\,|\,(r,s)\leq (i,j)\}\right|. \end{equation*}

So we have to count how many elements of \(S\) were already listed before reaching the element \((i,j)\) plus one for the latter.

It is convenient to split them by rows

\begin{align*} \mathrm{R}_1&=\{(1,2),\ldots,(1,n)\}\\ \mathrm{R}_2&=\{(2,3),\ldots,(2,n)\}\\ &\ldots\\ \mathrm{R}_{i-1}&=\{(i-1,i),\ldots,(i-1,n)\}\\ \mathrm{R}_i&=\{(i,i+1),\ldots,(i,j)\} \end{align*}

and then count the elements in each row:

\begin{align*} |\mathrm{R}_1|&=n-1,\\ |\mathrm{R}_2|&=n-2,\\ &\ldots\\ |\mathrm{R}_{i-1}|&=n-i+1,\\ |\mathrm{R}_i|&=j-i. \end{align*}

Therefore

\begin{align*} u&=|R_1|+|R_2|+\cdots+|R_{i-1}|+|R_i|\\ &=(n-1)+(n-2)+\cdots+(n-i+1)+(j-i)\\ &=\sum_{m=1}^{n-1}m-\sum_{m=1}^{n-i}m+j-i\\ &=\binom{n}{2}-\binom{n-i+1}{2}+j-i. \end{align*}
Lemma 3.1.4.

Let us consider the set

\begin{equation*} T=\{(i,j,k)\,|\,1\leq i<j<k\leq n\}. \end{equation*}

Then

\begin{equation*} |T|=\binom{n}{3} \end{equation*}

and if we list the elements of \(T\) in lexicographical order, we have

\begin{equation*} T=\{\underbrace{(1,2,3)}_{t_1},\underbrace{(1,2,4)}_{t_2},\ldots,\underbrace{(1,2,n)}_{t_{n-1}},\underbrace{(2,3,4)}_{t_n},\ldots,\underbrace{(i,j,k)}_{t_v},\ldots,\underbrace{(n-2,n-1,n)}_{t_{|T|}}\} \end{equation*}

with

\begin{equation*} v=\binom{n}{3}-\binom{n-i+1}{3}+\binom{n-i}{2}-\binom{n-j+1}{2}+k-j. \end{equation*}
Proof.

First observe that if we denote

\begin{equation*} (r,s,t)<(i,j,k) \end{equation*}

the lexicographical order, then

\begin{equation*} v=\left|\{(r,s,t)\in T\,|\,(r,s,t)\leq (i,j,k)\}\right|. \end{equation*}

So we have to count how many elements of \(T\) were already listed before reaching the element \((i,j,k)\) plus one for the latter.

  • If \(r=i\) and \(s=j\text{,}\) the elements to be taken into account are those for which

    \begin{equation*} j+1\leq t\leq k. \end{equation*}

    This yields

    \begin{equation*} A=k-j \end{equation*}

    elements.

  • If \(r=i\) and \(i<s<j\text{,}\) all the \(t\)'s such that

    \begin{equation*} s+1\leq t\leq n \end{equation*}

    must be taken into account, which yields

    \begin{equation*} n-s \end{equation*}

    elements.

    Therefore the case \(r=i\) and \(i<s<j\) yields

    \begin{align*} B&=(n-i-1)+(n-i-2)+\cdots+(n-j+1)\\ &=\sum_{m=1}^{n-i-1}m-\sum_{m=1}^{n-j}m\\ &=\binom{n-i}{2}-\binom{n-j+1}{2} \end{align*}

    elements.

  • For each \(r<i\text{,}\) all the pairs \((s,t)\) such that

    \begin{equation*} r+1\leq s<t\leq n \end{equation*}

    must be taken into account, which yields

    \begin{equation*} \binom{n-r}{2} \end{equation*}

    elements.

    Therefore the case \(r<i\) yields

    \begin{align*} C&=\binom{n-1}{2}+\cdots+\binom{n-i+1}{2}\\ &=\sum_{m=2}^{n-1}\binom{m}{2}-\sum_{m=2}^{n-i}\binom{m}{2}\\ &=\binom{n}{3}-\binom{n-i+1}{3} \end{align*}

    elements thanks to the hockey-stick identity.

Finally, we find

\begin{align*} v&=C+B+A\\ &=\binom{n}{3}-\binom{n-i+1}{3}+\binom{n-i}{2}-\binom{n-j+1}{2}+k-j. \end{align*}
