The column which corresponds to the following element of the start basis

\begin{equation*} x_s^*=x_{s_1}^*\wedge\cdots\wedge x_{s_{p-1}}^*\simeq s=[s_1,\ldots,s_{p-1}]\quad\text{where}\;1\leq s_1<\ldots<s_{p-1}\leq n \end{equation*}

is given by

\begin{equation*} \left[\delta^p(x_s^*)(x_e)\right] \end{equation*}

where

\begin{equation*} x_e=(x_{e_1},\ldots,x_{e_p}) \end{equation*}

with

\begin{equation*} e=[e_1,\ldots,e_p] \end{equation*}

running over the end basis.
