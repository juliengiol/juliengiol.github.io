First observe that if we denote

\begin{equation*} (r,s,t)<(i,j,k) \end{equation*}

the lexicographical order, then

\begin{equation*} v=\left|\{(r,s,t)\in T\,|\,(r,s,t)\leq (i,j,k)\}\right|. \end{equation*}

So we have to count how many elements of \(T\) were already listed before reaching the element \((i,j,k)\) plus one for the latter.
