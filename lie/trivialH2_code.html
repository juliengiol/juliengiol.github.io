The input consists of the structure constants stored in a NumPy array.

We will identify the vector

\begin{equation*} x_{k_1}^*\wedge\cdots\wedge x_{k_p}^* \end{equation*}

with the list

\begin{equation*} k=[k_1,\ldots,k_p]. \end{equation*}

Below is a function which lists the basis of \(\Lambda^p(\mathfrak{g}^*)\) in lexicographical order, given the above identification.

The matrix of

\begin{equation*} \delta^p:\Lambda^{p-1}(\mathfrak{g}^*)\longrightarrow\Lambda^p(\mathfrak{g}^*) \end{equation*}

with respect to the bases mentioned above is

\begin{equation*} \binom{n}{p}\times\binom{n}{p-1}. \end{equation*}

The column which corresponds to the following element of the start basis

\begin{equation*} x_s^*=x_{s_1}^*\wedge\cdots\wedge x_{s_{p-1}}^*\simeq s=[s_1,\ldots,s_{p-1}]\quad\text{where}\;1\leq s_1<\ldots<s_{p-1}\leq n \end{equation*}

is given by

\begin{equation*} \left[\delta^p(x_s^*)(x_e)\right] \end{equation*}

where

\begin{equation*} x_e=(x_{e_1},\ldots,x_{e_p}) \end{equation*}

with

\begin{equation*} e=[e_1,\ldots,e_p] \end{equation*}

running over the end basis.

Recall

\begin{equation*} \delta^p\alpha(v^1,\ldots,v^p)=\sum_{i<j}(-1)^{i+j}\alpha([v^i,v^j],v^1,\ldots,\cancel{v^i},\ldots,\cancel{v^j},\ldots,v^p) \end{equation*}

whence

\begin{align*} \delta^p(x_s^*)(x_e)&=\sum_{i<j}(-1)^{i+j}x_s^*([x_{e_i},x_{e_j}],x_{e_1},\ldots,\cancel{x_{e_i}},\ldots,\cancel{x_{e_j}},\ldots,x_{e_p})\\ &=\sum_{1\leq i<j\leq p}\sum_{l=1}^n(-1)^{i+j}c_{e_ie_j}^lx_s^*(x_l,x_{e_1},\ldots,\cancel{x_{e_i}},\ldots,\cancel{x_{e_j}},\ldots,x_{e_p}). \end{align*}

So we need to focus on every

\begin{equation*} x_s^*(x_l,x_{e_1},\ldots,\cancel{x_{e_i}},\ldots,\cancel{x_{e_j}},\ldots,x_{e_p}) \end{equation*}

whose value is either

\begin{equation*} -1,\quad 0\quad\text{or}\quad 1. \end{equation*}

Like in the previous chapter, the idea is to merge the index \(l\) in the increasing list

\begin{equation*} k = [e_1,\ldots,\cancel{e_i},\ldots,\cancel{e_j},\ldots,e_p] \end{equation*}

in order to put the list

\begin{equation*} lk = [l,e_1,\ldots,\cancel{e_i},\ldots,\cancel{e_j},\ldots,e_p] \end{equation*}

back in increasing order

\begin{equation*} kOut = [e_1,\ldots,l,\ldots,e_p] \end{equation*}

while keeping track of the sign.

Three cases are possible.

  1. If \(l\) is in the list \(e\text{,}\) then

    \begin{equation*} x_{lk}=0. \end{equation*}
  2. \begin{equation*} x_{lk}=x_{kOut}. \end{equation*}
  3. \begin{equation*} x_{lk}=-x_{kOut}. \end{equation*}

So we want a function merge(l, k) which returns

  1. \(0\) if \(l\) is in the list \(k\text{,}\)

  2. \((1, kOut)\) if \(x_{lk}=x_{kOut}.\)

  3. \((-1, kOut)\) if \(x_{lk}=-x_{kOut}.\)

Here is the function which did that in the previous chapter on homology.

Finally, when \(x_k\) is in increasing order, it is easy to compute \(x_s^*(x_k)\text{:}\)

\begin{equation*} x_s^*(x_k)=\begin{cases}1\text{ if }s=k\\0\text{ if }s\neq k\end{cases} \end{equation*}

since \(\{x_s^*\}\) is the dual basis of \(\{x_s\}\text{.}\) So we need a Kronecker symbol for lists.

We are ready to compute the coefficient

\begin{equation*} \delta^p(x_s^*)(x_e)=\sum_{1\leq i<j\leq p}\sum_{l=1}^n(-1)^{i+j}c_{e_ie_j}^lx_s^*(x_l,x_{e_1},\ldots,\cancel{x_{e_i}},\ldots,\cancel{x_{e_j}},\ldots,x_{e_p}). \end{equation*}

Be careful with the fact that we will have the correspondance

\begin{equation*} e[i]=e_{i+1} \end{equation*}

between the lists used when programming and the above formula.

Now we can build the matrix \(\Delta^p\) of the operator \(\delta^p\text{.}\)

That was the most difficult. Now we will use built-in functions of Sage to obtain a basis of

\begin{equation*} H^p(\mathfrak{g},\mathbb{C})=\mathrm{Ker}\delta^{p+1}\,/\,\mathrm{Im}\delta^p. \end{equation*}

Finally, we take this opportunity to define functions which compute the spaces of p-cocycles and p-coboundaries.
