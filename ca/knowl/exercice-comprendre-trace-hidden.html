Prendre la trace d'une surface 3D dans un plan \(z=k\text{,}\) c'est remplacer la variable \(z\) par la constante \(k\) dans l'équation qui la définit, puis étudier l'objet 2D obtenu.

On s'intéresse à la surface d'équation

\begin{equation*} z=x^2+y^2\text{.} \end{equation*}
  1. Décrivez la trace de cette surface dans le plan horizontal \(z=1\text{.}\)
  2. Décrivez la trace de cette surface dans le plan horizontal \(z=3\text{.}\)
  3. Décrivez la trace de cette surface dans le plan horizontal \(z=0\text{.}\)
  4. Décrivez la trace de cette surface dans le plan horizontal \(z=-2\text{.}\)
  5. Décrivez la trace de cette surface dans le plan vertical \(x=0\text{.}\)
  6. Décrivez la trace de cette surface dans le plan vertical \(y=0\text{.}\)
  7. Esquissez la représentation graphique de cette surface.
