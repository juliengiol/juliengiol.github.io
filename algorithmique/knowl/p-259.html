Exécutez l'algorithme de Horner à la main pour calculer \(P(3)\) si

  1. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&4\\ \text{Étape 1 :}&\text{Valeur = }&7+(-2)\cdot 4=-1\\ \text{Étape 2 :}&\text{Valeur = }&2+(-2)\cdot (-1)=4 \end{array} \end{equation*}

    Donc \(P(-2)=4\text{.}\)

  2. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&4\\ \text{Étape 1 :}&\text{Valeur = }&-7+(-2)\cdot 4=-15\\ \text{Étape 2 :}&\text{Valeur = }&-2+(-2)\cdot (-15)=28 \end{array} \end{equation*}

    Donc \(P(-2)=28\text{.}\)

  3. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&-5\\ \text{Étape 1 :}&\text{Valeur = }&0+(-2)\cdot(-5)=10\\ \text{Étape 2 :}&\text{Valeur = }&2+(-2)\cdot 10=-18\\ \text{Étape 3 :}&\text{Valeur = }&-1+(-2)\cdot(-18)=35 \end{array} \end{equation*}

    Donc \(P(-2)=35\text{.}\)
