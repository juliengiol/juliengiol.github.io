Exécutez l'algorithme de Horner à la main pour calculer \(P(3)\) si

  1. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&4\\ \text{Étape 1 :}&\text{Valeur = }&7+3\cdot 4=19\\ \text{Étape 2 :}&\text{Valeur = }&2+3\cdot 19=59 \end{array} \end{equation*}

    Donc \(P(3)=59\text{.}\)

  2. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&4\\ \text{Étape 1 :}&\text{Valeur = }&-7+3\cdot 4=5\\ \text{Étape 2 :}&\text{Valeur = }&-2+3\cdot 5=13 \end{array} \end{equation*}

    Donc \(P(3)=13\text{.}\)

  3. \begin{equation*} \begin{array}{lcl} \text{Étape 0 :}&\text{Valeur = }&-5\\ \text{Étape 1 :}&\text{Valeur = }&0+3\cdot(-5)=-15\\ \text{Étape 2 :}&\text{Valeur = }&2+3\cdot(-15)=-43\\ \text{Étape 3 :}&\text{Valeur = }&-1+3\cdot(-43)=-130 \end{array} \end{equation*}

    Donc \(P(3)=-130\text{.}\)
