Exercice 8.3.1.

Appliquez la méthode des rectangles à l'intégrale

\begin{equation*} \int_0^1\frac{1}{1+x^2}\mathrm{d}x \end{equation*}

avec cinq rectangles et donnez la valeur arrondie à trois décimales de l'approximation obtenue.

  1. En prenant les rectangles à gauche.

  2. En prenant les rectangles à droite.

  3. En prenant les rectangles au milieu.

Réponse
  1. \(\displaystyle \approx 0{,}834\)
  2. \(\displaystyle \approx 0{,}734\)
  3. \(\displaystyle \approx 0{,}786\)
Solution

Notons \(f(x)=\frac{1}{1+x^2}\text{,}\) \(n=5\) et

\begin{equation*} \Delta x=\frac{b-a}{n}=\frac{1-0}{5}=0{,}2. \end{equation*}

La subdivision régulière de l'intervalle d'intégration \([0;1]\) en cinq sous-intervalles de même longueur est donnée par

\begin{equation*} x_k=a+k\cdot\Delta x=k\cdot 0{,}2\quad\text{pour }k=0,1,\ldots,5, \end{equation*}

c'est-à-dire

\begin{equation*} 0<0{,}2<0{,}4<0{,}6<0{,}8<1. \end{equation*}

  1. Approximation par les rectangles à gauche :

    \begin{align*} &\Delta x\cdot\left(f(0)+f(0{,}2)+f(0{,}4)+f(0{,}6)+f(0{,}8)\right)\\ =\;&0{,}2\cdot\left(\frac{1}{1+0^2}+\frac{1}{1+0{,}2^2}+\frac{1}{1+0{,}4^2}+\frac{1}{1+0{,}6^2}+\frac{1}{1+0{,}8^2}\right)\\ \approx\;&0{,}834. \end{align*}

  2. Approximation par les rectangles à droite :

    \begin{align*} &\Delta x\cdot\left(f(0{,}2)+f(0{,}4)+f(0{,}6)+f(0{,}8)+f(1)\right)\\ =\;&0{,}2\cdot\left(\frac{1}{1+0{,}2^2}+\frac{1}{1+0{,}4^2}+\frac{1}{1+0{,}6^2}+\frac{1}{1+0{,}8^2}+\frac{1}{1+1^2}\right)\\ \approx\;&0{,}734. \end{align*}

  3. Approximation par les rectangles au milieu :

    \begin{align*} &\Delta x\cdot\left(f(0{,}1)+f(0{,}3)+f(0{,}5)+f(0{,}7)+f(0{,}9)\right)\\ =\;&0{,}2\cdot\left(\frac{1}{1+0{,}1^2}+\frac{1}{1+0{,}3^2}+\frac{1}{1+0{,}5^2}+\frac{1}{1+0{,}7^2}+\frac{1}{1+0{,}9^2}\right)\\ \approx\;&0{,}786. \end{align*}
Exercice 8.3.2.

Appliquez la méthode des rectangles à l'intégrale

\begin{equation*} \int_0^\pi\sin(x)\mathrm{d}x \end{equation*}

avec quatre rectangles et donnez la valeur arrondie à trois décimales de l'approximation obtenue.

  1. En prenant les rectangles à gauche.

  2. En prenant les rectangles à droite.

  3. En prenant les rectangles au milieu.

Réponse
  1. \(\displaystyle \approx 1{,}896\)
  2. \(\displaystyle \approx 1{,}896\)
  3. \(\displaystyle \approx 2{,}052\)
Solution

Notons \(f(x)=\sin(x)\text{,}\) \(n=4\) et

\begin{equation*} \Delta x=\frac{b-a}{n}=\frac{\pi-0}{4}=\frac{\pi}{4}. \end{equation*}

La subdivision régulière de l'intervalle d'intégration \([0;\pi]\) en quatre sous-intervalles de même longueur est donnée par

\begin{equation*} x_k=a+k\cdot\Delta x=k\cdot\frac{\pi}{4}\quad\text{pour }k=0,1,\ldots,4, \end{equation*}

c'est-à-dire

\begin{equation*} 0<\frac{\pi}{4}<\frac{\pi}{2}<\frac{3\pi}{4}<\pi. \end{equation*}

  1. Approximation par les rectangles à gauche :

    \begin{align*} &\Delta x\cdot\left(f(0)+f\left(\frac{\pi}{4}\right)+f\left(\frac{\pi}{2}\right)+f\left(\frac{3\pi}{4}\right)\right)\\ =\;&\frac{\pi}{4}\cdot\left(\sin(0)+\sin\left(\frac{\pi}{4}\right)+\sin\left(\frac{\pi}{2}\right)+\sin\left(\frac{3\pi}{4}\right)\right)\\ \approx\;&1{,}896. \end{align*}

  2. Approximation par les rectangles à droite :

    \begin{align*} &\Delta x\cdot\left(f\left(\frac{\pi}{4}\right)+f\left(\frac{\pi}{2}\right)+f\left(\frac{3\pi}{4}\right)+f(\pi)\right)\\ =\;&\frac{\pi}{4}\cdot\left(\sin\left(\frac{\pi}{4}\right)+\sin\left(\frac{\pi}{2}\right)+\sin\left(\frac{3\pi}{4}\right)+\sin(\pi)\right)\\ \approx\;&1{,}896. \end{align*}

  3. Approximation par les rectangles au milieu :

    \begin{align*} &\Delta x\cdot\left(f\left(\frac{\pi}{4}\right)+f\left(\frac{\pi}{2}\right)+f\left(\frac{3\pi}{4}\right)+f(\pi)\right)\\ =\;&\frac{\pi}{4}\cdot\left(\sin\left(\frac{\pi}{8}\right)+\sin\left(\frac{3\pi}{8}\right)+\sin\left(\frac{5\pi}{8}\right)+\sin\left(\frac{7\pi}{8}\right)\right)\\ \approx\;&2{,}052. \end{align*}
Exercice 8.3.3.

Appliquez la méthode des rectangles à l'intégrale

\begin{equation*} \int_2^3x\,\mathrm{d}x \end{equation*}

avec dix rectangles et donnez la valeur exacte de l'approximation obtenue.

  1. En prenant les rectangles à gauche.

  2. En prenant les rectangles à droite.

  3. En prenant les rectangles au milieu.

Réponse
  1. \(\displaystyle 2{,}45\)
  2. \(\displaystyle 2{,}55\)
  3. \(\displaystyle 2{,}5\)
Solution

Notons \(f(x)=x\text{,}\) \(n=10\) et

\begin{equation*} \Delta x=\frac{b-a}{n}=\frac{3-2}{10}=0{,}1. \end{equation*}

La subdivision régulière de l'intervalle d'intégration \([2;3]\) en dix sous-intervalles de même longueur est donnée par

\begin{equation*} x_k=a+k\cdot\Delta x=2+k\cdot 0{,}1\quad\text{pour }k=0,1,\ldots,10, \end{equation*}

c'est-à-dire

\begin{equation*} 2<2{,}1<2{,}2<\ldots<2{,}9<3. \end{equation*}

Comme il y a beaucoup de valeurs, nous allons utiliser la notation sigma pour exprimer l'approximation cherchée :

\begin{align*} \text{Approximation}&=\Delta x\cdot\sum_{k=1}^nf(\xi_k)\\ &=0{,}1\cdot\sum_{k=1}^{10}\xi_k. \end{align*}

  1. Rectangles à gauche : On prend \(\xi_k=x_{k-1}\) donc

    \begin{align*} \text{Approximation}&= 0{,}1\cdot\sum_{k=1}^{10}(2+(k-1)\cdot 0{,}1)\\ &=0{,}1\cdot\left(\sum_{k=1}^{10}2+\left(\sum_{k=1}^{10}(k-1)\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(\sum_{k=1}^{10}2+\left(\sum_{k=1}^9k\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(10\cdot 2+\left(\frac{9\cdot(9+1)}{2}\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(20+45\cdot 0{,}1\right)\\ &=2{,}45. \end{align*}

  2. Rectangles à droite : On prend \(\xi_k=x_k\) donc

    \begin{align*} \text{Approximation}&=0{,}1\cdot\sum_{k=1}^{10}(2+k\cdot 0{,}1)\\ &=0{,}1\cdot\left(\sum_{k=1}^{10}2+\left(\sum_{k=1}^{10}k\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(10\cdot 2+\left(\frac{10\cdot(10+1)}{2}\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(20+55\cdot 0{,}1\right)\\ &=2{,}55. \end{align*}

  3. Rectangles à droite : On prend \(\xi_k=\frac{x_{k-1}+x_k}{2}=2+(k-0{,}5)\cdot 0{,1}\) donc

    \begin{align*} \text{Approximation}&=0{,}1\cdot\sum_{k=1}^{10}(2+(k-0{,}5)\cdot 0{,}1)\\ &=0{,}1\cdot\left(\sum_{k=1}^{10}2+\left(\sum_{k=1}^{10}k-\sum_{k=1}^{10}0{,}5\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(10\cdot 2+\left(\frac{10\cdot(10+1)}{2}-10\cdot 0{,}5\right)\cdot 0{,}1\right)\\ &=0{,}1\cdot\left(20+55-5\cdot 0{,}1\right)\\ &=2{,}5. \end{align*}
Exercice 8.3.4.

Encadrez la valeur de l'intégrale

\begin{equation*} \int_0^1\frac{1}{1+x}\mathrm{d}x \end{equation*}

en appliquant la méthode des rectangles à gauche avec cinq rectangles et en utilisant trois décimales.

Réponse
\begin{equation*} 0{,}645\leq\int_0^1\frac{1}{1+x}\mathrm{d}x\leq 0{,}846 \end{equation*}
Solution

  • Notons \(f(x)=\frac{1}{1+x}\text{.}\)

  • Soit \(n=5\) et \(\Delta x=\frac{b-a}{n}=\frac{1-0}{5}=0{,}2\text{.}\)

  • Alors \(x_k=a+k\cdot\Delta x=k\cdot 0{,}2\) pour \(k=0,\ldots,5\text{.}\)

  • La subdivision régulière de l'intervalle \([0;1]\) à considérer est donc la suivante :

    \begin{equation*} 0<0{,}2<0{,}4<0{,}6<0{,}8<1. \end{equation*}

  • Par la méthode des rectangles à gauche, on obtient

    \begin{align*} \text{Approximation}&=0{,}2\cdot\left(f(0)+f(0{,}2)+f(0{,}4)+f(0{,}6)+f(0{,}8)\right)\\ &=0{,}2\cdot\left(1+\frac{1}{1{,}2}+\frac{1}{1{,}4}+\frac{1}{1{,}6}+\frac{1}{1{,}8}\right)\\ &=0{,}745\ldots \end{align*}

    On a donc

    \begin{equation*} 0{,}745\leq\text{Approximation}\leq 0{,}746. \end{equation*}

  • Notons que \(|f'(x)|=\frac{1}{(1+x)^2}\text{.}\) Pour tout \(x\in[0;1]\text{,}\) on a

    \begin{align*} &&1+x&\geq 1\\ &\Rightarrow&(1+x)^2&\geq 1\\ &\Rightarrow&\frac{1}{(1+x)^2}&\leq 1. \end{align*}

    Donc \(|f'(x)|\leq 1\) pour tout \(x\in[0;1]\) et on peut prendre \(M=1\) dans le théorème 8.1.11.

  • On a donc

    \begin{align*} |\text{Erreur}|&\leq\frac{M(b-a)^2}{2n}\\ &=\frac{1\cdot(1-0)^2}{2\cdot 5}\\ &=0{,}1 \end{align*}

    d'où

    \begin{equation*} -0{,}1\leq\text{Erreur}\leq 0{,}1. \end{equation*}

  • En additionnant les inéquations qui précèdent, on obtient

    \begin{equation*} 0{,}745-0{,}1\leq\text{Approximation}+\text{Erreur}\leq 0{,}746+0{,}1 \end{equation*}

    c'est-à-dire

    \begin{equation*} 0{,}645\leq\int_0^1\frac{1}{1+x}\mathrm{d}x\leq 0{,}846. \end{equation*}

Remarque : Sachant que cette intégrale vaut \(\ln 2=0,693\ldots\text{,}\) on comprend qu'il faudrait beaucoup plus de rectangles pour être capable d'en déterminer les premières décimales par cette méthode.

Exercice 8.3.5.

On souhaite approcher l'intégrale définie

\begin{equation*} \int_0^1\frac{1}{1+x}\mathrm{d}x \end{equation*}

à l'aide de la méthode des rectangles.

Combien faudrait-il utiliser de rectangles au minimum pour être certains que la valeur absolue de l'erreur d'approximation ne dépasse pas \(10^{-6}\text{?}\)

Réponse

\(500\,000\)

Solution

On a déjà vu à l'exercice précédent qu'on pouvait appliquer le théorème 8.1.11 avec \(M=1\text{.}\)

En d'autres termes, on cherche le plus petit \(n\) tel que

\begin{equation*} \begin{array}{cccc} &|\text{Erreur}|&\leq&10^{-6}\\ \Leftrightarrow&\frac{M(b-a)^2}{2n}&\leq&10^{-6}\\ \Leftrightarrow&\frac{1\cdot(1-0)^2}{2n}&\leq&10^{-6}\\ \Leftrightarrow&\frac{1}{2n}&\leq&10^{-6}\\ \Leftrightarrow&\frac{10^6}{2}&\leq&n\\ \Leftrightarrow&500\,000&\leq&n \end{array} \end{equation*}

Il faudrait donc \(500\,000\) rectangles au minimum.

Exercice 8.3.6.

Modifiez le code donné à la section 8.2 afin d'appliquer la méthode des rectangles à droite à l'intégrale

\begin{equation*} \int_0^14\sqrt{1-x^2}\,\mathrm{d}x=\pi \end{equation*}

pour \(1000\) rectangles.

Vous devez trouver \(\approx 3{,}139\,555\,466\,911\,027\,7\text{.}\)

Solution
# entrées
import numpy as np
def f(x):
    return 4 * np.sqrt(1 - x ** 2)
a = 0.0
b = 1.0
n = 1000

# instructions
approximation = 0
deltaX = (b - a) / n
x = b
while x > a:
    approximation = approximation + f(x)
    x = x - deltaX
approximation = deltaX * approximation

# sortie
print(approximation)
Exercice 8.3.7.

Modifiez le code donné à la section 8.2 afin de remplacer la boucle while par une boucle for semblable à celle utilisée à la section 9.2.

Solution
# entrées
import numpy as np
def f(x):
    return 4 * np.sqrt(1 - x ** 2)
a = 0.0
b = 1.0
n = 32

# instructions
approximation = 0
deltaX = (b - a) / n
for k in range(1, n+1):
    x = a + (k - 1) * deltaX
    approximation = approximation + f(x)
approximation = deltaX * approximation

# sortie
print(approximation)
Exercice 8.3.8.

Créez une fonction rectanglesAuMilieu(f, a, b, n) qui donne l'approximation de l'intégrale \(\int_a^bf(x)\mathrm{d}x\) par la méthode des rectangles au milieu pour \(n\) rectangles.

Testez votre code pour \(\int_0^1\frac{1}{1+x}\mathrm{d}x\) avec \(1000\) rectangles. Vous devez trouver \(\approx 0{,}693\,147\,149\,31\text{.}\)

Solution
# fonction
def rectanglesAuMilieu(f, a, b, n):
    approximation = 0
    deltaX = (b - a) / n
    x = a + deltaX / 2
    while x < b:
        approximation = approximation + f(x)
        x = x + deltaX
    approximation = deltaX * approximation
    return approximation

# test
def f(x):
    return 1 / (1 + x)
print(rectanglesAuMilieu(f, 0.0, 1.0, 1000))
