Solution 5.3.3.1.

Il suffit de déplacer print(x0) à l'intérieur de la boucle for :

# entrées :
def f(x):
    return x ** 2 - 2
def fPrime(x):
    return 2 * x
x0 = 2.0
n = 3
# instructions / sortie :
for i in range(1, n+1):
    x0 = x0 - f(x0) / fPrime(x0)
    print(x0) # instruction déplacée
