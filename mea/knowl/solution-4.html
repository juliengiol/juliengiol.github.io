  1. Il s'agit des 5 nombres qui suivent.

    7.0
1.23529411765
0.224598930481
0.0415923945336
0.00779857397504

    On peut les afficher à l'aide du bloc de code suivant.

    n = 3
x = 7.0
while n < 8:
    print(x)
    x = x * n / (5 * n + 2)
    n = n + 1

    n = 3
