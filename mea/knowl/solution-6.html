Solution 1.4.10.1.

En partant de \(14\text{,}\) les 21 premiers termes de cette suite sont

\begin{equation*} 14, 7, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1, 4, 2, 1. \end{equation*}

Notez qu'une fois qu'on est tombé sur \(1\text{,}\) le comportement de la suite devient périodique.

La célèbre conjecture de Syracuse (ou Collatz, ou Ulam, ou 3x+1, ...) prédit qu'une telle suite finit toujours par tomber sur \(1\text{,}\) quel que soit l'entier positif dont on parte. Mais personne n'a encore réussi à le démontrer.

On peut les afficher à l'aide du bloc de code suivant.

s = 14
for n in range(1, 21+1):
    print(s)
    if s % 2 == 0:
        s = s / 2
    else:
        s = 3 * s + 1
