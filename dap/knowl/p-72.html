Profitez-en pour noter que si vous voulez utiliser une esperluette (&) ou un plus petit que (<) dans un fichier ptx pour autre chose que ce à quoi ils servent en XML (le langage utilisé dans les fichiers ptx), vous devrez les remplacer par &amp; et &lt;. Ceci se produit typiquement quand on entre du code LaTeX tel que

\begin{equation*} A=\begin{bmatrix}1&2\\3&4\end{bmatrix} \end{equation*}

ou \(x<5\text{.}\)
